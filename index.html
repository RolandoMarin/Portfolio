Project Overview

The Health and Fitness Tracker app project aims to address the pervasive challenges associated with sedentary lifestyles, poor dietary habits, and inadequate fitness management by leveraging technology to empower users to take control of their health and wellness goals. The task involved developing a comprehensive mobile application that seamlessly integrates features such as workout tracking, meal logging, goal setting, progress monitoring, and social engagement within an intuitive and user-friendly interface.

Project Background

This project was created by Rolando Marin-Galvan, and was created for my Capstone project for Grand Canyon University. This application was made in the span of two classes. First iteration involved planning, and design choices. The second part of the class involved writing, and testing code. This final project demonstrates the skills and knowledge earned over the past few years with Grand Canyon University.

Design Details

The general technical approach for developing the Health and Fitness Tracker app revolves around a structured and iterative process that encompasses various stages from requirement analysis to deployment and maintenance. I started by brainstorming sessions to gather project requirements, define user stories, and identify key features such as workout tracking, meal logging, goal setting, social engagement, and personalized coaching. The design incorporates a modular and scalable architecture, utilizing Flutter for cross-platform UI development, Firebase for backend services and data storage, and third-party integrations for additional functionalities. The application will prioritize security measures such as encryption, secure authentication, and data privacy to ensure user trust and compliance with regulations.

To run the application, simply download the repo and install the dependencies within the README File and run the project. The app is currently not live due to having to purchase a yearly license to upload to Google play and Apple App Store.

User Guide

Login: Open App -> Tap Login Button -> Enter Email and password Fields -> Tap Submit -> Success

Register: Open App -> Tap Register Button -> Enter Email and password Fields -> Tap Submit -> Success

Create Post: Open App -> Follow Login Steps-> Go to home -> Tap plus button -> Enter post details -> Success

Like a Post: Open App -> Login steps -> Go to Home -> Tap on heart icon on selected post -> Success

Unlike a Post: Open App -> Login steps -> Go to Home -> Tap on red heart icon on selected post -> Success

Delete a post: Open App -> Login Steps -> Go to home -> Tap on three dots top right of your own post -> Tap Delete Post -> Success

Update a post: Open App -> Login Steps -> Go to home -> Tap on three dots top right of your own post -> Tap Edit Post -> Success

Log a meal: Open App -> Login Steps -> Go to Log -> Tap meals tab -> Tap plus button -> Enter fields to log meal -> Success

Log a Workout: Open App -> Login Steps -> Go to Log -> Tap workout tab -> Tap plus button -> Enter fields to log workout -> Success

Update a Log: Open App -> Login Steps -> Go to Log -> Select tab -> Click on edit or Delete button to choose -> Success

Update Account: Open App -> Login Steps -> Go to Account -> Tap on selected update -> Success
